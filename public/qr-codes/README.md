# QR Code Images

This directory contains QR code images for stations 1-5.

## Files

- `qr-s01.png` - QR code for station_id 1 (s01)
- `qr-s02.png` - QR code for station_id 2 (s02)
- `qr-s03.png` - QR code for station_id 3 (s03)
- `qr-s04.png` - QR code for station_id 4 (s04)
- `qr-s05.png` - QR code for station_id 5 (s05)

## Usage

Each QR code contains the station ID as plain text (e.g., "s01", "s02", etc.).

To generate more QR codes, run:
```bash
node scripts/generate-qr-codes.mjs
```

## Download

These images are available in the public folder and can be accessed at:
- `/qr-codes/qr-s01.png`
- `/qr-codes/qr-s02.png`
- `/qr-codes/qr-s03.png`
- `/qr-codes/qr-s04.png`
- `/qr-codes/qr-s05.png`
