# Images Directory

## Required Images

### village-street.jpg
Please add the village street scene image (the colorful medieval/fantasy timber-frame buildings) to this directory as `village-street.jpg`.

The image should show:
- Colorful buildings in medieval/fantasy timber-frame style
- Light green, yellow, blue, and orange-red buildings
- Cobblestone street
- Trees in the background
- Clear blue sky

This image will be displayed at the top of the welcome/explanation page.
